# 📊 A/B Testing para optimización de conversión en e-commerce

## 🎯 Objetivo
Evaluar si una nueva versión del sitio web mejora la conversión de usuarios.

## 🧠 Problema de negocio
Determinar si la versión B del sitio genera mejores resultados que la versión A.

## 🛠 Herramientas
- Python (Pandas, NumPy, Matplotlib)
- SQL
- Estadística (Mann-Whitney U)

## 🔍 Proceso
- Limpieza de datos
- Priorización con ICE y RICE
- Análisis exploratorio
- Identificación de outliers
- Pruebas estadísticas

## 📈 Insights clave
- Grupo B mejora la conversión en ~15%
- Diferencia estadísticamente significativa (p < 0.05)
- No hay cambio en el ticket promedio

## 💰 Impacto
- Incremento potencial en ingresos
- Mejora en decisiones de negocio

## 🚀 Conclusión
Se recomiendaimplementarla versión B.
